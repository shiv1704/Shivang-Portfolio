# QUICK TIP ON DECK LINKS

Before sharing: make sure each deck is set to "Anyone with the link can view" — otherwise visitors will hit permission walls.

- **Google Slides:** Share → General access → "Anyone with the link" → Viewer
- **Figma:** Share → "Anyone with the link" → can view
- **Notion:** Share → Publish to web → copy link
- **PDFs:** Upload to your repo's `/public/decks/` folder, link as `/decks/filename.pdf`
